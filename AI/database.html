  • テーブルからデータを抽出したい。
    SELECT[取得したい要素] FROM [使用テーブル];
  • 実行結果の重複レコードを１つにまとめたい。
    SELECT DISTINCT 列名 列名 FROM テーブル名;
  • 特定の条件を満たすレコードを取り出す。「グループ化される前に、特定ユーザーを省く」
    SELECT フィールド名 FROM テーブル名 WHEWE 条件式;
  • サブクエリ
    SQL文の中に入れ子になってSQL文が書いていること。
  • 合計を求めたい。
    SELECT SUM(得点) FROM 試験結果;
  • 平均を求めたい。
    SELECT SUM(得点) FROM 試験結果;
  • 数を数えたい
    SELECT COUNT(得点) FROM 試験結果;
  • 指定したグループの最大値を取得したい。
    SELECT MAX(抜き出すカラム) AS カラム名 FROM テーブル名;
  • 指定したグループの最小値を取得したい。
    SELECT MIN(抜き出すカラム) AS カラム名 FROM テーブル名;
  • 種類ごとに集計関数を利用する。例えばチームとの人数を数える。GROUP BY
